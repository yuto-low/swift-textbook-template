## この教科書について

この教科書は、iOSプログラミングIIの授業を通じて、自分自身の手で作り上げるSwiftプログラミングの教科書です。

毎回の授業で教員から渡される模範コードを題材に、「このコードはなぜこう書かれているのか」を自分の言葉で説明することで、コードを深く理解することを目的としています。
